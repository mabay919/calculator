Как работает калькулятор?

  • Просто очищает строку!
  • "√" - вводим нужное число => нажимаем √ => получаем корень с введенного числа. Также можно получить корень с выведенного на экран результата вычислений.
  • "/"" - вводим делимое число нажимаем / вводим число делитель нажиамем "=" и получаем результат!
  • "*" - умножение порядок действий как в предыдущем примере.
  • "**" - вводим число, которое хотим поднести к нужной степени => нажимаем ** => вводим значение степени => нажимаем = и получаем результат.
  • "+" - действуем аналогично предыдущим пунктам.
  • "-" - действуем аналогично предыдущим пунктам.
  • "-/+" - если нужно ввести отрицательное число => нажимаем "-/+" => вводим нужное число.
  • Очищает введенную строку и память (полная очистка)!
